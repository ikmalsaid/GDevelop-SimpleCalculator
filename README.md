# GDevelop-SimpleCalculator
A basic calculator made with GDevelop and JavaScript

## Features
* Number buttons (0 to 9).
* Equal, decimal and clear buttons.
* Add, subtract, multiply and divide buttons.
* Math and octal error handling.
